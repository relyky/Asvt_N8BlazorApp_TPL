拖曳分隔線調整區塊大小
左上區域

這是左上半分割畫面

使用 CSS Grid 實作的三區塊分割佈局。

佈局說明

  • 外層使用 Grid 進行左右分割
  • 左側再使用 Grid 進行上下分割
  • 右側為單一完整區域

你可以拖曳中間的分隔線來調整各區塊的大小。

左下區域

這是左下半分割畫面

此區塊與左上區塊共享左側空間，可透過水平分隔線調整比例。

技術特點

  • Grid Template Columns 控制左右比例
  • Grid Template Rows 控制上下比例
  • 動態調整 grid 尺寸
  • 最小尺寸保護機制
